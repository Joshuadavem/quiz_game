##### Quiz Game



###### This program runs a trivia game where the player is prompted to answer questions and the player types in the answer, it will then give you a final score.





## Features:



 - Randomized question order

 - User input

 - Answer checking

 - Case-insensitive answers

 - Score tracking

 - End-of-game feedback







## How to Run



Open the folder

Open the python file

Run and debug it

Play in the terminal
